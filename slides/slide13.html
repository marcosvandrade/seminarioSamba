                # mkdir /home/.../publico

                Mudando as permissões da pasta publico para acesso total
                # chmod -R 2777 /home/.../publico

                Caso seja necessário, verifique as configurações atuais do samba executando
                o comando abaixo.
                $ testparm

                Abra o gerenciador de arquivos e tente acessar o compartilhamento
                smb://127.0.0.1
